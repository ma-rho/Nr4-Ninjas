# **App Name**: NR4 Ninjas

## Core Features:

- Event Listing: Display a list of upcoming events with details like date, location, and description. Integrates with Firestore for data storage and retrieval.
- Ticket Link Integration: Link events to external ticketing platforms (Fatsoma / Event link) for easy ticket purchases.
- Merch Shop: Display clothing product listings with images, sizes, and prices, stored in Firebase.
- Stripe Checkout: Enable secure Stripe payment processing for merchandise purchases.
- Photo Gallery: Display event photos organized by event, pulling assets from Firebase Storage. Optimizes image loading for performance.
- DJ Profiles: Display individual DJ profiles with names, bios, photos, social links, and booking CTAs, using data stored in Firestore.
- Newsletter Signup: Gathers customer emails to enable email marketing.
